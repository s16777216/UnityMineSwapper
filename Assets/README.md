# Unity Minesweeper

## Use story

1. User啟動遊戲，進入`主畫面`
2. User點擊`開始遊戲`按鈕，進入`遊玩場景`
3. User點擊`方格`，觸發方格開啟事件
4. 觸發`遊戲勝利事件`，遊戲結束，顯示遊戲`遊戲勝利畫面`
5. 觸發`遊戲失敗事件`，遊戲結束，顯示遊戲`遊戲失敗畫面`
6. User點擊`重新開始`按鈕，重載`遊玩場景`
7. User點擊`回到主畫面`按鈕，回到`主畫面`
8. User點擊`回到桌面`按鈕，關閉遊戲

## Use case

### 1. User啟動遊戲，進入`主畫面`

1. User啟動遊戲
2. 顯示`加載畫面`
3. 初始化遊戲資料
4. 進入`主畫面`

### 2. User點擊`開始遊戲`按鈕，進入`遊玩場景`

1. User點擊`開始遊戲`按鈕
2. 顯示`加載畫面`
3. 初始化遊戲資料
4. 進入`遊玩場景`

### 3. User點擊`方格`，觸發方格開啟事件

1. User點擊`方格`
2. 觸發方格開啟事件
3. 顯示`方格內容`
4. 判斷是否為`地雷`
5. 若為`地雷`，觸發`遊戲失敗事件`
6. 若不為`地雷`，繼續遊戲
7. 若符合`遊戲結束條件`，觸發`遊戲勝利事件`

### 4. 觸發`遊戲勝利事件`，遊戲結束，顯示遊戲`遊戲勝利畫面`

1. 觸發`遊戲勝利事件`
2. 顯示`遊戲勝利畫面`
3. 顯示`遊戲結果`

### 5. 觸發`遊戲失敗事件`，遊戲結束，顯示遊戲`遊戲失敗畫面`

1. 觸發`遊戲失敗事件`
2. 顯示`遊戲失敗畫面`
3. 顯示`遊戲結果`

### 6. User點擊`重新開始`按鈕，重載`遊玩場景`

1. User點擊`重新開始`按鈕
2. 顯示`加載畫面`
3. 初始化遊戲資料
4. 進入`遊玩場景`

### 7. User點擊`回到主畫面`按鈕，回到`主畫面`

1. User點擊`回到主畫面`按鈕
2. 顯示`加載畫面`
3. 進入`主畫面`

### 8. User點擊`回到桌面`按鈕，關閉遊戲

1. User點擊`回到桌面`按鈕
2. 關閉遊戲
